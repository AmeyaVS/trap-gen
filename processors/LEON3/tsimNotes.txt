Command Line for TSIM with no floating point unit, with 8 MB of normal memory, with hardware breakpoints, with no ROM

tsim-leon -fast_uart -hwbp -logfile +tsimOutLog -nfp -nosram -nouart -ram 8192
To start simulation go 0x40000000 0

Using the BCC compiler, the compiltion options are
-msoft-float -mv8
in order to use soft floating point and the SPARC multiplication and division instructions
